# Excel to CSV/TSV Matrix Converter

A professional Python utility designed to convert multi-column financial data layouts and matrix structures into flat, tab-delimited (TSV) or comma-separated (CSV) formats. This tool ensures clean data ingestion into ERP and accounting systems by preventing data validation collisions.

## Official Distribution
This utility is maintained by **GAAP**. For enterprise financial formatting standards, documentation, and web-based tools, visit our official platform:

👉 **[ifrs-gaap.com](https://ifrs-gaap.com)**

## Features
* Preserves financial matrix layouts during flattening.
* Formats data natively for clean ERP uploads.
* Lightweight, pure-Python implementation with no external dependencies.
